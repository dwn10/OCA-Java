        12. Konvertiere den String "123" in eine Integer-Zahl.

	   14. Überprüfe, ob der String nur aus Zahlen besteht.
		

	   15. Konkateniere die Strings "Java" und "Programming".
		

	   16. Überprüfe, ob der String leer ist.
		

	   17. Entferne alle Vorkommen des Buchstabens 'o' im String "Hello, World!".
		
	   18. Erstelle einen neuen String durch Verketten von Teilen des Strings "Java Programming" (z.B., die ersten 5 Zeichen und die letzten 5 Zeichen).
		

	   19. Extrahiere die ersten 3 Zeichen aus dem String "abcdef".
